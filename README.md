# Trivago MCP App (WIP)

This is a work-in-progress Laravel + Vue application exploring how LLM-driven travel search experiences can be built using the trivago MCP API.

## Current Focus
- Integrating OpenAI for natural language query processing
- Connecting to the trivago MCP API for accommodation search
- Structuring the application using a service-based architecture

## Tech Stack
- Laravel (backend)
- Vue.js (frontend)
- OpenAI API
- Trivago MCP API

## Status
Early-stage project focused on architecture and API integration. Further development will include UI improvements, filtering, and richer search experiences.
